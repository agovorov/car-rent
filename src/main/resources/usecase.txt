Task:
15.	Система Прокат автомобилей. Клиент выбирает Автомобиль из списка доступных.
    Заполняет форму Заказа, указывая паспортные данные, срок аренды. Клиент оплачивает Заказ.
    Администратор регистрирует возврат автомобиля. В случае повреждения Автомобиля,
    Администратор вносит информацию и выставляет счет за ремонт. Администратор может
    отклонить Заявку, указав причины отказа.

Админ
    + Управляет (CRUD) списком авто на портале
    + Редатирует справочники(производители, цвета, кузов и т.д.)
    + Выдает авто (ставит отметку)
    + Потдверждает заказ
    + Проверяет оплату
    + Фиксирует возврат авто
    + Выписывает счет за ремонт (при необходимости)

Клиент
    + Просматривает список доступных авто
    + Выбирает одну машину
    + Авторизуется (Регистрируется)
    + Заполняет заявку на аренду авто
    + Ожидает подтверждения заявки
    + Оплачивает полную стоимость
    + Возвращает авто
    + При наличии повреждений, оплачивает выставленный счет
